# C Singly Linked List Implementation

A modular implementation of a Singly Linked List in C, featuring standard operations like insertion, deletion, and reversing.

## Features
* **Append/Prepend**: Add nodes to the start or end.
* **Insert at Position**: Add nodes at specific indices.
* **Delete**: Remove specific nodes or clear the entire list.
* **Reverse**: In-place reversal of the list logic.

## How to Compile
Use `gcc` to compile the library and the main driver:
```bash
gcc main.c list.c -o linked_list
./linked_list
